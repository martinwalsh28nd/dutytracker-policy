# dutytracker-policy

Public home for the DutyTracker privacy policy. Served as a GitHub
Pages site so App Store Connect's **Privacy Policy URL** field has
something to point at.

This repo exists as a split-off from `martinwalsh28nd/work-hour-logger`
because that repo is private and GitHub Pages on private repos
requires GitHub Pro. A tiny standalone public repo is the cheaper
alternative.

## Updating

Edit `privacy-policy.md`, commit, push. GitHub Pages rebuilds within
~60 seconds. The live URL stays stable.

The **source of truth** for the policy lives in
`work-hour-logger/docs/privacy-policy.md`. Any substantive edits
should happen there and be mirrored here.
